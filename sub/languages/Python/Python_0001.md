|数値型（int型）|整数リテラル|100|

### 数値型

|型|表現内容|
----|----
|int型|整数リテラル|
|float型|浮動小数点数リテラル|

#### 整数リテラル

##### 10進数の書き方

数字をそのまま記述

##### 2,8,16進数の書き方

先頭に数字の0、2文字目に特定のアルファベットを付し、その後にそれぞれの方法で表現された数字を記述  
2文字目に付すアルファベットは以下の通り

|数値の表現方法|2文字目に付すアルファベット|
----|----
|2進数|b又はB|
|8進数|0又はO|
|16進数|x又はX|

##### 扱える整数の範囲

+ 最小値：-9,223,372,036,854,775,808
+ 最大値：9,223,372,036,854,775,807
